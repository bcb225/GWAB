GWAB configuration
  1. There are some settings user can choose.
  2. Some settings are hardwired for user convenience.
GWAB configuration set
Meaning Example
Disease Name User-defined disease name Coronary Artery Disease
Build Version Human genome build default : Hg18
Residue up/down distance to find nearby genes default : 10000
Log P Value Range of p-value threshold for GWAB default : -6 ~ -2
Interval Step of p-value threshold default : 0.3
Suffles The number of suffles to generate random distribution Hardwired : 10
FPR False Positive Rate for GWAB validation Hardwired : 0.05
